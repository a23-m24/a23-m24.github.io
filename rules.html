Правила игры

Игра назвается "Lights out!". Изначально вам дается поле размером NxN клеток(размер поля выбирается в настройках). Каждое из клеток поля закрашено в один из двух цветов. При нажатии на клетку весь ряд и колонна, в которых находится клетка меняет свой цвет на противоположный. Для победы необходимо собрать все поле в один из двух цветов.

Лучшие результаты по времени прохождения игры и по количеству нажатий сохраняются в специальной сранице. Результаты сохраняются для каждого из доступных размера поля.

В настройках можно выбрать:

  • N - размер поля
  • Фигуру, из которой будет сосотоять поле
  • Темную или светлую тему оформления страницы
